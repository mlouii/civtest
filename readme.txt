# Civ-Like MVP (Python + Pygame)

A minimal Civilization-inspired strategy game foundation, built with Python and Pygame.  
This project is for rapid prototyping and future integration of AI/LLM agents.

---

## 🏗️ Features

- 2D grid-based map with randomized terrain (grassland, plains, forest, hill, mountain, water)
- Core classes for Map, Tile, and (soon) Unit/City logic
- Turn-based structure
- Pygame-based graphical rendering of the map grid
- Expandable for units, cities, production, and more
- Unit tests for foundational logic (run with `python test_game.py`)

---

## 🚀 Quickstart

**Prerequisites:**  
- Python 3.10+  
- [Pygame](https://www.pygame.org/news) (`pip install pygame`)

```bash
